# CatchBook

Mon projet porte sur l’activité des pêcheurs. Je souhaite mettre en place une bibliothèque regroupant les poissons en général, les cannes à pêche à utiliser, ainsi que différentes techniques de pêche.
L’application devra permettre aux utilisateurs d’ajouter les poissons qu’ils ont vus ou attrapés, de préciser le lieu où ils les ont pêchés, d’ajouter des photos de leurs prises, ainsi que d’indiquer leur taille et leur poids approximatif.
Les utilisateurs pourront également commenter les publications.
